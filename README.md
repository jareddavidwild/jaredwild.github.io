# jaredwild.github.io
This personal website serves as a my digital hub, acting as an intersection between a professional portfolio, an updated resume, and a personal blog. Unlike social media profiles, this platform is 100% owned, allowing me to control the narrative, design, and content. The site is designed to be a "digital handshake".

Made with [three]([https://pages.github.com/](https://github.com/mrdoob/three.js/)) along with [fireship](https://www.youtube.com/watch?v=Q7AOvWpIVHU)
